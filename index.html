Mini Documentation

  • .click()

    Bind an event handler to the "click" event or trigger that eventon an element. Example: An alert would pop up that an event has been triggered.

  • .hide()

    Hide the matched elements. Example: Hiding the H2 Element

  • .show()

    Display the matched elements. Example: Showing the H2 Element that was hidden before.

  • .toggle()

    Display or hide the matched elements. Example: Toggling H2 as being shown or hidden

  • .slideUp()

    Hide the matched elements with a sliding motion. Example: hiding the H2 Element while sliding up

  • .slideDown()

    Display the matched elements with a sliding motion. Example: showing the H2 Element while sliding down

  • .slideToggle()

    Display or hide the matched elements with a sliding motion. Example hiding or showing the H2 Element with a sliding motion

  • .fadeOut()

    Hide the matched elements by fading them to transparent. Example: Hiding the H2 Element by making it transparent

  • .fadeIn()

    Display the matched elements by fading them to opaque. Example: Showing the H2 Element by making it opaque

  • .addClass()

    Adds the speicified class(es) to each element in the set of matched elements. Example: adding a CSS class in the H2 element

  • .before()

    Insert content, specified by the parameter, before each element in the set matched elements. Example: Adding a text before H2.

  • .after()

    Insert content, specified by the parameter, after each element in the set of matched elements. Example: Adding a text after H2

  • .html()

    Get the HTML contents of the first element in the set of matched elements or the set HTML contents of every matched element. Example: adding a text with HTML elements to the H2 element

  • .attr()

    Get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element. Example: Changing the attribute of the image below

  • .val()

    Get the current value of the first element in the set of matched elements or the set value of every matched element. Example: Adding value inside an input

  • .text()

    Get the combined text contents of element in the set of matched elements, including their descendants, or set the text contents of the matched elements. Example: Changing the text of the H2 Element
